# V1 产品范围

## 目标

轻仓是团队共用库存的微信原生小程序。管理员维护产品和库存，普通成员搜索和查看库存。

## 阶段0已做

- 建立微信原生小程序基础架构。
- 建立 CloudBase 初始化入口。
- 建立本地用户启动初始化骨架。
- 建立四个底部导航页。
- 建立基础组件、服务层、工具层和常量层。
- 建立仓库管理器独立云函数 `warehouse-api` 骨架与 `system.ping`。
- 建立数据设计、权限和开发规范文档。

## 阶段0暂不做

正式产品列表、产品新增编辑、产品搜索、入库出库、库存调整、真实团队创建、邀请成员、微信头像昵称授权、图片上传、数据统计图表、正式 UI 还原、多仓库、采购单、销售单、供应商、Excel 导入导出和订阅消息。
