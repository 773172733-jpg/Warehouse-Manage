# 口袋仓库Go 候选版本更新说明

> 建议候选版本号：`1.2.0`（等待用户根据微信后台当前线上版本确认）

## 极短版本描述

完善商品、库存流水与预警管理，优化型号搜索和团队只读权限体验。

## 微信上传版本说明

本次候选版本完善商品创建、编辑、回收与恢复，支持系统贴图、文字封面和自定义图片；支持入库、出库、盘点、商品型号模糊搜索、单商品及全仓库存流水，并提供低库存和缺货预警。团队成员按角色使用，viewer保持只读。同步完成稳定性与发布前质量检查。

## 详细用户更新日志

### 商品管理

- 支持创建、编辑、回收和恢复商品。
- 商品封面可使用系统贴图、文字封面或用户主动选择的图片。
- 优化商品型号模糊搜索，可通过型号中间片段更快找到商品。

### 库存管理

- 支持入库、出库和盘点，并保留操作前后库存变化。
- 商品详情可查看对应商品的永久库存流水。
- 底部“记录”可统一查看全仓库存流水。
- 提供低库存与缺货预警，帮助及时识别需要补货的商品。

### 团队协作

- 支持团队成员角色管理。
- owner和admin可执行相应管理操作，viewer可安全查看商品、流水和预警。
- 优化新用户创建团队、输入邀请码和等待审核的状态提示。

### 体验与稳定性

- 移除正式页面中的演示数据和未实现占位入口。
- 统一用户可见产品名称与隐私说明。
- 完成全量自动化测试、页面注册、配置、编码和静态检查。

本说明只描述当前已经实现并通过自动化检查的功能。建议在上传体验版前完成开发者工具编译、iOS与Android真机、弱网及各角色的人工回归。
