# 权限矩阵

## owner

- 团队创建者
- 管理管理员
- 管理普通成员
- 管理共享产品、仓库实例和库存
- 全局删除与恢复共享目录产品
- 团队最高权限

## admin

- 新增和编辑产品
- 入库
- 出库
- 调整库存
- 从当前仓库移除和恢复零库存产品
- 查看记录

## viewer

- 查看产品
- 搜索产品
- 查看库存
- 不允许任何写操作

## 权限控制原则

前端隐藏按钮不等于权限控制。所有写操作后续必须由云函数查询 `team_members` 验证，不允许信任前端传入的 `role`、`teamId`、`openId` 或 `userId`。

V1页面中的”删除产品”表示软移除当前 `warehouse_products`，不等于删除团队共享 `products`。全局目录删除与恢复仅owner可执行，定于2C3B实现对应UI，不阻塞2C2。
